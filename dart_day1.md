```dart

    print('你好 dart');

```

## 创建变量 `type key = value;`

>var自动推断变量类型
>明确类型定义 String,int等
>final 和const 定义常量

## dart 的命名规则

>字母下划线数字美元符号组成
>不能为关键字
>不能数字开头
>大小写有区分
>见名思意

## 运算符

>运算符`+ - / *  % ~/`其中~/表示取整

## 方法

> 返回类型 名称 (参数， [可选参数]) {}
